Welcome to Lena Contingencies Version 3

For Users:

How to Run

  1. Download Lena Contingencies from GitHub! The application is located in ./builds/v3
  2. Run program from executable located in that folder.
  3. Setup input directory(for batch of .its or .csv files)
  4. Setup output directory
  5. Configure Analysis
  6. Press Submit!

For Developers:

How to Build the Program:

Windows: See: windows_cmds.txt in the root directory for full instructions on setting up pyinstaller and using the provided spec file to update the build for future developments.

MacOS & Unix: To build the MacOS and Unix versions of the application for future releases, please install pyinstaller. Once installed, execute the python script located in ./buildScript/build_MacUnix.py with python build_MacUnix.py This will bundle the app and place the executables in ./builds/v3
